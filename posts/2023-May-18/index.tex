\hypertarget{choices}{%
\subsection{Choices}\label{choices}}

I tried several options:

\begin{itemize}
\item
  Static site generators, Jekyll-based options and Python scripts
\item
  Simple html files using Notepad++ to build a website to hold my notes
\end{itemize}

Then I'm back to .md files using Zettlr, because it supports:

\begin{itemize}
\item
  Math typesetting i.e.~good \(\LaTeX\) support \(\LaTeX\)
\item
  Easy input of more commands in .md than in .html using Notepad++
\item
  Easy .html and .pdf generation
\item
  Citation built-in with Zotero
\end{itemize}

But I discovered \href{https://quarto.org}{Quarto} which is a scientific
and technical publishing system utilizing
\href{https://pandoc.org}{Pandoc}. It does everything Zettlr does but in
style. But Zettlr is still best for raw note-taking.
